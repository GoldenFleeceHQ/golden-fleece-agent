Deep Research Prompt: Golden Fleece — AI Trading Agent (ERC-8004 Hackathon)

CONTEXT

We are building a trustless AI trading agent for the AI Trading Agents with ERC-8004 Hackathon (March 9–22, 2026, $50K prize pool). The agent must register on the ERC-8004 Identity Registry, accumulate reputation from trading outcomes, produce validation artifacts for every key action, and execute trades through a whitelisted Risk Router contract on Base Sepolia. Judging emphasizes risk-adjusted returns (Sharpe ratio), drawdown control, validation quality, and presentation — not raw PnL.

Our proposed stack is Python (AI/trading logic) + TypeScript (on-chain integration and Next.js dashboard), using Claude Opus as the primary LLM, viem for contract interactions, and Foundry for Solidity tooling.

Please perform deep research on each of the following 12 areas. For each area, provide: (1) Key Findings, (2) Specific Recommendations for our project, (3) Risks and Unknowns, and (4) Links and Resources. Prioritize actionable specificity — we need concrete answers (contract addresses, API endpoints, code examples), not just overviews. Flag anything where our assumptions might be wrong or outdated.


AREA 1: ERC-8004 Standard — Current State & Implementation Reality

- What is the current status of ERC-8004 (EIP-8004)? Is it Draft, Review, or Final? What are the latest changes to the spec?
- Examine the ERC-8004 contracts repo (https://github.com/erc-8004/erc-8004-contracts): What contracts are deployed on Base Sepolia? What are their addresses? Are there known bugs or limitations?
- Examine the awesome-erc8004 repo (https://github.com/sudeepb02/awesome-erc8004): What JS/TS SDK tooling exists? How mature is it? Are there Python bindings?
- Read the Ethereum Magicians discussion (https://ethereum-magicians.org/t/erc-8004-trustless-agents/25098) for any gotchas, open questions, or implementation notes from the authors.
- Check 8004.org for any official documentation, tutorials, or example agents.
- Are there any existing open-source agents that have already integrated ERC-8004 that we can learn from?
- What is the exact flow for: (a) registering an agent identity, (b) logging a validation artifact, (c) submitting reputation feedback? Walk through each step including contract calls, required parameters, and gas considerations on Base Sepolia.
